# Computer Architecture - VHDL Projects

A collection of VHDL implementations developed during my Computer Architecture coursework. This repository demonstrates the design, simulation, and implementation of digital systems using Intel FPGA development tools.

---

## Overview

This repository contains projects that explore the fundamentals of digital design and computer architecture through Hardware Description Language (VHDL).

The objective is to demonstrate practical experience in:

- Digital Logic Design
- Combinational Circuits
- Sequential Circuits
- Register Transfer Level (RTL) Design
- Finite State Machines (FSM)
- Arithmetic Logic Units (ALU)
- FPGA Development
- Hardware Verification through Simulation

## Getting Started

### Clone the repository

```bash
git clone https://github.com/<your-username>/Computer-Architecture-VHDL.git
```

### Open a project

1. Launch Intel Quartus Prime.
2. Open the desired project.
3. Compile the design.
4. Program the FPGA (if applicable).
5. Run simulations using Questa or ModelSim.
